# Python Foundations Assignment

## 📌 Overview

This repository contains solutions for Python foundational tasks across three levels.

## ✅ Tasks Implemented

### Level 1: Prime Number Checker

- Checks whether a number is prime using an optimized approach.

### Level 2: Fibonacci Sequence

- Generates Fibonacci sequence up to n terms.

### Level 3: CLI Calculator

- A command-line calculator supporting:
  - Addition
  - Subtraction
  - Multiplication
  - Division

## 🚀 How to Run

```bash
python level1_prime.py
python level2_fibonacci.py
python level3_calculator.py
```
